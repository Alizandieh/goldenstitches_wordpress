== Description ==

WP Disables Updates allow you to disables plugin or themes or wordpress core updates.

Major features :

* Disable plugins updates.
* Disable themes updates.
* Disable Wordpress core updates.

PS: Used there features if you know what you are doing. No updates = Security holes, but sometimes updates could break your website.

[Freelance Wordpress / Prestashop](https://vincent-guesne.fr)

== Installation ==

Upload or install Wp Disable Updates on wp-content/plugins/ then go to settings > Disables updates > Check option.

That's all folks
